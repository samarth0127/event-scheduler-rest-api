Event Scheduler REST API

A powerful and simple REST API for managing scheduled events with full CRUD operations, file-based persistence, and comprehensive search functionality.

Manage Events View Reminders API Documentation Learn More

Key Features

Create Events

Create events with title, description, start time, and end time. Input validation ensures data integrity.

List & Sort

Retrieve all events sorted chronologically by start time. Get detailed information for each event.

Update Events

Modify existing events with partial or complete updates. Flexible API supports updating any event field.

Delete Events

Remove events permanently from the system. Safe deletion with proper error handling.

Search Events

Find events by searching through titles and descriptions. Fast and efficient text-based search.

File Persistence

Data persists between sessions using JSON file storage. No database setup required - works out of the box.

API Endpoints

Method Endpoint Description
GET /api/events Get all events (sorted by start time)
GET /api/events/{id} Get a specific event by ID
POST /api/events Create a new event
PUT /api/events/{id} Update an existing event
DELETE /api/events/{id} Delete an event
GET /api/events/search?q={query} Search events by title or description
